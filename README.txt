Expected inputs and outputs:
Once, code is running, the animated scene will be created
You can increase or decrease the speed using UP or DOWN arrow keys respectively
You can click on the object to revolve around itself, along verical axis and click on same object to stop
You can toggle lights usinh 1,2,3,4 keys
You can select the objects using A,B,C,D and then control that object.
In controlling an object you can scale it, translate it to where ever you click right mouse button.
Press T to cycle through textures(Marble,Checkerboard,Map,Lena)
Press M to cycle through mappings(Cylindrical,Spherical,Normal)

Defects:
If there are 2 objects overlapping, pickpoint is not exact
